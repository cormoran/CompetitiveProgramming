# 問題概要

 １を根とする木が与えられる。A,BがAから交互に次数１の頂点を取っていく。頂点１を取ったほうが勝ち。両方が最善を尽くした時、どっちが勝つ？

# 解法

　1の子うち、一つを除いて残りをすべて取り除かないとAは取れない。子を取り除くにはその子を根とする部分木をすべて取り除かないといけない。
