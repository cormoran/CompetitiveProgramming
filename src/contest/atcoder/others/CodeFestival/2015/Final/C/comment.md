# 問題概要

+ abを追加
+ baを追加
+ a or bを追加して残りを取っておく、取っておいたのはいつでも追加できる

# 解法

  貪欲っぽい問題でよくわからないのは、とりあえず貪欲書いてみてそのあとじっくり反例とか考えるのが良さそうだと毎回思うけどできてない...
